# ends.at

A calm place to paste Markdown, publish it, and share the page.

> Beautiful text should not need a dashboard around it. ends.at keeps the page quiet, spacious, and readable, using native system fonts that already feel at home on every device.

## Share the good part

AI tools are great at producing useful drafts, summaries, plans, specs, recipes, study notes, release notes, and tiny essays.

ends.at gives that output a clean URL.

```markdown
# Launch notes

- What changed
- Why it matters
- What to do next
```

No account flow. No theme picker. No heavy editor chrome. Just the response, rendered as a page worth reading.

## What it feels like

| You have | ends.at gives you |
| --- | --- |
| A ChatGPT response | A readable page you can send |
| A rough Markdown draft | A clean preview and publish link |
| Notes in a Google Sheet | A simple public page for each row |
| Text that deserves space | Native fonts, generous rhythm, fewer distractions |

## Made for lightweight publishing

1. Paste Markdown.
2. Preview it immediately.
3. Publish a short link.
4. Share the result.

That is the whole idea.

## A small example

### Project brief

Build a tiny tool that makes writing feel less trapped.

- Keep the page fast.
- Let Markdown do the structure.
- Make long text comfortable to read.
- Make sharing feel almost invisible.

> The product is not the editor. The product is the moment when text becomes easy to pass along.
