# About ends.at

ends.at is a small place for turning Markdown into clean, shareable pages.

It is built for the moment when you already have the words:

- an AI response worth keeping
- a note you want to send
- a draft that reads better outside a chat window
- a list, plan, brief, recipe, summary, or tiny essay

Paste the text, publish it, and share the page.

## Why it exists

So much useful writing now starts somewhere temporary: a chat, a note, a sheet, a scratch file.

ends.at gives that text a simple home on the web without asking you to design a website around it.

> The goal is not to make a heavier editor. The goal is to make sharing text feel almost invisible.

## What it values

| Value | What that means here |
| --- | --- |
| Quiet | Fewer controls, less visual noise. |
| Readable | Native fonts, generous spacing, comfortable line length. |
| Fast | Paste, preview, publish. |
| Portable | Markdown in, clean page out. |
| Shareable | Every published page has a URL. |

## Markdown

Markdown is a plain-text way to add structure to writing: headings, links, quotes, lists, code, and tables.

You do not need to know much Markdown to use ends.at. If your text came from an AI tool, a note app, GitHub, or a Google Sheet, it may already be Markdown.

Curious? Read more at [markdownguide.org](https://www.markdownguide.org/).

## Google Sheets

ends.at can also render Markdown from a public Google Sheet.

- One filled cell becomes one page.
- Multiple rows use column A for Markdown.
- Column B can define the slug for each row.

Start with [/sheet](/sheet).

## Power shortcuts

For people who like moving quickly:

| Shortcut | Action |
| --- | --- |
| `Cmd + Enter` | Publish from the editor on Mac. |
| `Ctrl + Enter` | Publish from the editor on Windows/Linux. |
| `E` | Edit a published page when you are not typing in a field. |
| `/` | Open the More menu when you are not typing. Use arrow keys and `Enter` to choose an item. |
| `Esc` | Close the menu, or exit editing back to the published page. |

The shortcuts are intentionally few. The page should still feel like a page.
