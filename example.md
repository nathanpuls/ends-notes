# Brief: The Anti-Meeting Machine

![A clean workspace with notes and a laptop](https://picsum.photos/seed/brief-machine/1200/600)

> What if half the meetings on your calendar were just pages that explained themselves?

This is a short Markdown example about a fictional tool that turns messy updates into clean, shareable briefs. It includes **bold text**, *italic text*, `inline code`, [a useful link](https://www.markdownguide.org/), and enough structure to show how a page can breathe.

---

## The Pitch

The Anti-Meeting Machine takes a pile of notes and returns one readable page:

1. Paste the raw update.
2. Shape it into Markdown.
3. Publish the page.
4. Send the link.

### Why teams like it

* fewer status calls
* clearer decisions
* better handoffs
* writing that survives the chat scroll
